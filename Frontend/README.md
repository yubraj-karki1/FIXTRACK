# FixTrack

FixTrack is a Next.js App Router app built with React, TypeScript, and Tailwind/global CSS.

## Run app

```bash
npm run dev
```

The app runs on `http://localhost:3000`.

The frontend uses the same-origin `/api` rewrite by default. Configure `BACKEND_URL` for the Next.js server rewrite or `NEXT_PUBLIC_API_URL` for a separate browser-visible API origin.
